# Reimu 科研记录博客：本地第一阶段设计

## 目标

为 GitHub 用户 `damin93381` 建立一个本地可运行的 Hexo 博客，主题采用官方 Reimu 模板。博客用于长期记录计算化学、材料科学、技术实践与日常生活。

第一阶段只交付本地站点和预览能力；不创建 GitHub 仓库、不推送远程分支、不配置 GitHub Pages 或 GitHub Actions。

## 技术方案

从官方 `D-Sketon/reimu-template` 初始化项目。该模板包含 Hexo、Reimu 主题及常用依赖，能够最大程度保留主题的原始视觉、布局和交互。

站点在本机通过 Hexo 开发服务器预览；构建命令生成静态站点，以便在未来发布前作最终验证。

## 信息架构与初始内容

- 站点标题：`damin93381`
- 副标题：`计算化学、材料科学与日常记录`
- 栏目：科研笔记、技术实践、日常记录、关于
- 社交链接：GitHub 主页 `https://github.com/damin93381`
- 关于页：介绍博客聚焦的研究与记录范围；不虚构个人履历、单位或联系方式。
- 初始文章：一篇开篇文章，说明本站用于沉淀计算化学、材料科学、工具实践和日常思考。

内容分类通过文章 front matter 提供，保留 Reimu 默认的文章、分类、标签、归档、搜索和深浅色等原生能力。

## 主题与自定义边界

第一阶段不重做样式、不替换 Reimu 的设计语言、不引入视觉上显著的新组件。仅更改站点元数据、导航、侧边栏资料、社交链接和内容，以使默认主题能承载科研记录。

头像、封面图和其他个人素材在未提供前使用模板默认资源；不从外部抓取或臆造个人图像。

## 验收标准

1. 依赖能够安装，`hexo generate` 可成功构建静态站点。
2. 本地开发服务器可启动并呈现 Reimu 原生样式。
3. 首页、关于页和开篇文章可访问，三类文章分类可用。
4. 页面内不包含尚未填写的虚构个人资料。
5. 工作区不含任何远程发布或 GitHub Actions 部署配置。

## 后续阶段（不在本次实施范围）

在用户完成本地预览并明确授权后：创建 `damin93381/damin93381.github.io`、配置 GitHub Pages 自动部署、推送站点，并按需要持续调整视觉和内容。
