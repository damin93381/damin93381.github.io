# 幽幽子本地预览图像包设计

## 目标

仅为本地运行的 Reimu 科研博客替换指定图片，形成以西行寺幽幽子为主题的预览效果。站点结构、功能、文字内容、颜色令牌、发布状态和远程配置均不改变。

## 已选素材角色

| 站点位置 | 选定候选 | 本地用途 |
| --- | --- | --- |
| 全站横幅 | 横幅候选 #6：月夜、天鹅与幽幽子（Pixiv 127200026，作者标注为 zzz / orchid-dale） | 从同一源图裁切桌面与移动端横幅，确保角色姿势一致。 |
| 全站横幅备选 | 横幅候选 #12：鸟居晚霞 | 仅保留来源记录，不启用。 |
| 侧栏头像 | 头像候选 #6：绯想天官方立绘 | 制作正方形头像裁切。 |
| 文章封面 | 封面候选 #12：Touhou Genso Eclipse 樱花卡面 | 生成默认本地文章封面。 |
| 文章封面备选 | 封面候选 #11、#8、#3 | 仅保留来源记录，不启用。 |
| 404 背景 | 404 候选 #1：伞、樱花与灵蝶（shirokuro，Pixiv 108644218） | 替换仅在 404 页面出现的背景插画。 |
| 小图标、光标 | 不变 | favicon、页脚、回到顶部、预加载和鼠标指针保持原样。 |

## 文件与裁切

- 桌面横幅：WebP，2560×1200；标题安全区在角色相反侧，占画面约 35%。
- 移动横幅：WebP，1440×1080；从与桌面相同的原图独立裁切。
- 头像：WebP，600×600；保留脸部、粉发和帽檐。
- 默认文章封面：WebP，2400×1350（16:9）。
- 404 背景：WebP，2560×1440；角色置于一侧，另一侧保留错误文案空间。

## 本地预览与授权边界

所选图片均没有确认的公开再发布许可。它们只用于用户明确授权的本地预览，且每项都记录来源页面、原作者（若已知）和“公开发布前需要取得许可或替换为原创／获授权素材”的状态。

本阶段不得创建远程仓库、推送、部署、GitHub Pages 或 GitHub Actions 配置。未来发布前，必须移除这些预览素材或获得适当许可。

## 验收标准

1. 本地横幅、头像、文章封面和 404 页面使用指定图片角色。
2. 桌面和移动横幅来自同一幅原图，角色姿势一致。
3. favicon、页脚、回到顶部、预加载和鼠标指针不变。
4. npm test 和本地预览正常。
5. 项目中有可查阅的来源与授权状态记录，且无发布配置。
